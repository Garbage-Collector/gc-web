
# Garbage Collector

국립한밭대학교 정보통신공학과 2024 캡스톤 디자인 웹 프론트엔드 레포지터리 입니다.


## 설치 방법
아래의 단계로 수행하시면 됩니다.

### 1. node 설치


<b>asdf</b>
```
asdf install nodejs 20.11.1
asdf global nodejs 20.11.1
```

<b>nvm</b>
```
nvm install 20.11.1
nvm use
```

### 2. 패키지매니저 활성화(yarn, pnpm)
```
corepack enable
```

### 3. 의존성 설치 및 실행
#### 3-1. pnpm
<b>pnpm</b>
```
pnpm i
pnpm run dev
```

#### 3-2. yarn
<b>yarn</b>
```
yarn set version 4.1.1
yarn
yarn dev
```
